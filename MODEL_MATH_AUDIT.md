# Математический аудит моделей

Дата: 14 июля 2026 года.

## Результат

Команды `node scripts/audit-math.mjs` и `node scripts/audit-models.mjs` проходят. Проверены 9 игр, 78 призовых уровней в каждой из двух HTML-версий, 8 427 текущих тиражей и 108 фактических прогонов двенадцати базовых моделей.

| Модель | Реализованная математика | Статус |
|---|---|---|
| Частотный анализ | последовательный выбор без возвращения с весом `c(n)+0,05` | исправлено и проверено |
| Сбалансированный | около 40% горячих, 30% средних, 30% холодных; округление чередуется | исправлено и проверено |
| Pure random | криптографическая перестановка допустимых чисел без возвращения | проверено |
| Сегментный охват | по одному числу из каждого непересекающегося сегмента диапазона | проверено |
| Цепи Маркова | `P(b|a)=(Nab+1)/(Na+N)`, усреднение по числам последнего тиража | исправлено и проверено |
| Гаусс · ЦПТ | `μ=k(N+1)/2`, `σ²=k(N+1)(N−k)/12`; при базе — эмпирические сумма и σ | проверено |
| Интервальная модель Δ | bootstrap первого числа и соседних разностей отсортированных тиражей | переименовано точно и проверено |
| Байес · Дирихле | апостериорные веса `c(n)+α`, симметричный prior `α=3` | проверено |
| Gap-анализ | эвристика `(gap/E[gap])^1,6+0,15`; не трактуется как «число обязано выпасть» | проверено |
| Физическая модель | упрощённые 2D ускорение/drag/стенки/выход и упругий нормальный импульс равных масс | переписано и проверено |
| Детерминированный хаос | логистическое отображение `x→3,99x(1−x)` с криптографическим seed | проверено |
| Квантовый коллапс | ANU QRNG, если доступен, иначе `crypto.getRandomValues`; экспоненциальный частотный барьер | исправлен источник потока и проверено |

Структурное поле, Консенсус и Судья используют общий набор частот, lift пар, пропусков, энтропии, сумм и разброса ряда. Исправлены ключи истории `main/bonus`, из-за которых ранее часть баллов могла становиться нулевой.

## База результатов

- `results.json`: 8 427 тиражей, совместимых с текущими правилами; только этот файл загружается приложением.
- `results-archive.json`: 13 159 записей для офлайн-исследований, включая старые форматы с меткой `legacy`.
- Vikinglotto расширен до 9 июня 2021 года — первого розыгрыша после последнего старого формата 2 июня.
- NY Open Data загружается с пагинацией; Mega Millions в исследовательском архиве начинается 17 мая 2002 года.
- EuroMillions в исследовательском архиве расширен до первого тиража 13 февраля 2004 года; старые диапазоны Lucky Stars не смешиваются с текущим форматом.
- Диагностика ищет будущие даты, некорректные числа и подозрительные календарные разрывы. Разрыв SuperEnalotto 21.03–05.05.2020 сохранён как реальный период остановки, а не заполнен выдуманными данными.

## Ограничения

Исторические частоты не повышают вероятность независимой комбинации. «Без повторов» в симуляторе — продуктовый режим: сохраняемая перестановка проходит пространство `C(N,k)` без дублей до конца цикла. Реальная лотерея может повторить прошлую комбинацию. История исследовательских моделей ограничена 20 000 выданных главных комбинаций на игру из-за лимита локального хранилища устройства.
