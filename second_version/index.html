χ² тест проверяет равномерность распределения.

p > 0.05 → лотерея честная ✅
p < 0.05 → отклонение ⚠️
p < 0.01 → сильное отклонение 🚨

Даже «нечестная» лотерея не даёт предсказания — каждый тираж независим.
